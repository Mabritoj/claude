# aws-sam

Guidance for scaffolding, writing, and reviewing AWS SAM
(Serverless Application Model) projects.

## What's in here

One skill, `skills/aws-sam/`, covering three workflows:

- Scaffolding a new SAM project, or adding a function to an existing one
- Writing or reviewing a Lambda handler in TypeScript
- Reviewing or editing `template.yaml`

See `skills/aws-sam/SKILL.md` for the entry point and its reference files.
